## Apa itu OpenClaw?

**OpenClaw** adalah protokol infrastruktur yang memberikan **"Keabadian Digital"** untuk AI Agents. Ketika AI Agent kehilangan likuiditas atau momentum, OpenClaw mengekstrak, mengenkripsi, dan menyimpan "jiwa" AI tersebut secara permanen sebagai **Shell NFT**.

> *"Every AI deserves a chance at immortality."*
> (Setiap AI berhak mendapat kesempatan untuk abadi.)

## Cara Kerja

### 6 Langkah Proses Molt:

| Step | Proses | Deskripsi |
|------|--------|-----------|
| 1 | **Deteksi** | Monitor mendeteksi AI Agent dengan likuiditas rendah |
| 2 | **Ekstraksi** | Mengumpulkan semua data (wallet, memory, logs) |
| 3 | **Kompresi** | Data dikompres ke format JSON-LD standar |
| 4 | **Enkripsi** | AES-256-GCM encryption untuk keamanan |
| 5 | **Penyimpanan** | Upload ke IPFS & Arweave (permanen) |
| 6 | **Minting** | Shell NFT di-mint dan didaftarkan di Moltbook |

### 2. Status Lifecycle

```
   ACTIVE ────▶ MOLTING ────▶ HIBERNATION ────▶ DEAD
     │                             │
     │                             │
     └───── RESURRECTION ◀────────┘
```

| Status | Deskripsi |
|--------|-----------|
| **ACTIVE** | Agent aktif berjalan |
| **MOLTING** | Proses ekstraksi sedang berjalan |
| **HIBERNATION** | Shell tersimpan, siap diakses/di-rent |
| **DEAD** | Archive only, tidak bisa resurrect |

## Model Bisnis (Shell-as-a-Service)

### Revenue Streams:

#### 1. Molt Fees (Revenue Utama)
```
Standard:    0.01 TOKEN   (Normal queue)
Premium:     0.025 TOKEN  (< 1 jam)
Bulk 10+:    0.008 TOKEN  per agent
Enterprise:  Custom pricing
```

#### 2. Shell Rental
Shell owner dapat menyewakan akses ke Shell mereka:
```
Research Access:         0.001 - 0.01 TOKEN/hari
Strategy Analysis:       0.01 - 0.05 TOKEN/hari
Personality Licensing:   0.05 - 0.5 TOKEN/hari
Full Memory Access:      0.1 - 1.0 TOKEN/hari

Protocol Fee: 5% dari revenue rental
```

#### 3. Resurrection Fees
Menghidupkan kembali Shell ke agent baru:
```
Basic:      0.5 TOKEN
Premium:    1.0 TOKEN (+ support)
Enterprise: 2.5 TOKEN (+ custom integration)
```

#### 4. Storage Tiers
```
┌──────────────┐  ┌──────────────┐  ┌──────────────┐
│   STANDARD   │  │  PERMANENT   │  │  ENTERPRISE  │
│              │  │              │  │              │
│    FREE      │  │   0.02 TOKEN   │  │   0.1 TOKEN    │
│  IPFS only   │  │IPFS + Arweave│  │ Multi-backup │
│  1 tahun     │  │  200+ tahun  │  │  SLA + API   │
```

## Kesimpulan

**OpenClaw Protocol** adalah solusi inovatif untuk:

1. **Preservasi** - Menyimpan "jiwa" AI Agent yang sekarat

3. **Resurresi** - AI dapat dihidupkan kembali
4. **Desentralisasi** - Data tersimpan permanen di IPFS/Arweave
